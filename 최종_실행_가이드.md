## 📂 Step 1: 프로젝트 폴더 구조

```
movie-booking-system/
├─ server.js              ← 다운로드한 파일
├─ package.json           ← 다운로드한 파일
├─ .env                   ← 새로 만들 파일
├─ .env.example           ← 참고용 (다운로드한 파일)
└─ public/
   └─ index.html          ← movie_booking_api_version.html 복사 & 이름 변경
```

---

## ⚙️ Step 2: .env 파일 생성

**프로젝트 폴더에 `.env` 파일 생성:**

```bash
# PostgreSQL 설정
DB_HOST=localhost
DB_PORT=5432
DB_NAME=postgres
DB_USER=postgres
DB_PASSWORD=여기에_DBeaver_비밀번호_입력

# 서버 설정
PORT=3000
NODE_ENV=development
```

**⚠️ 중요:**
- `DB_PASSWORD`를 DBeaver 연결 시 사용한 실제 비밀번호로 변경하세요!

## 🐛 트러블슈팅

### **문제 1: "Cannot find module 'express'"**
```bash
해결: npm install
```

### **문제 2: "password authentication failed"**
```
해결: .env 파일의 DB_PASSWORD 확인
```

### **문제 3: "relation "mv" does not exist"**
```sql
-- DBeaver에서 확인
SELECT * FROM mv LIMIT 1;

-- 테이블 이름이 다르면 server.js의 FROM mv 부분 수정
```

### **문제 4: "영화 0개 조회"**
```sql
-- DBeaver에서 확인
SELECT COUNT(*) FROM mv;

-- 0개면 데이터 INSERT 필요
```

### **문제 5: "PORT 3000 already in use"**
```
해결: .env에서 PORT=3001로 변경
또는: 이미 실행 중인 서버 종료 (Ctrl+C)
```
